# Firewall Types Overview

## Objective
To understand different types of firewalls and how they help protect networks and systems.

## What I Learned

1. Software Firewalls
Software firewalls are installed on individual devices and help monitor and control incoming and outgoing traffic on that specific system.

2. Hardware Firewalls
Hardware firewalls are physical devices placed between a network and the internet to filter traffic for multiple devices.

3. Cloud-Based Firewalls
Cloud firewalls provide firewall protection through cloud services and are commonly used in modern cloud environments.

## Stateful vs Stateless Firewalls

== Stateful Firewalls
Stateful firewalls monitor active connections and analyze traffic based on the state of the connection.

== Stateless Firewalls
Stateless firewalls filter traffic based only on predefined rules without tracking connection states.

## Next-Generation Firewalls (NGFWs)

NGFWs provide advanced security features beyond traditional firewalls, including:
- Deep packet inspection
- Application awareness
- Intrusion prevention features
- Advanced threat detection

## Key Takeaway
Different firewall types provide different levels of protection depending on network requirements. Advanced firewalls such as NGFWs offer stronger security capabilities for modern threats.
